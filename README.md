# EduQuery AI

## Project Title
EduQuery AI – Smart Student Support System for Universities with Multi-Platform Access

## Description
EduQuery AI is a chatbot designed to help students with queries like exams, fees, admission, and library.

## Features
- Chatbot support
- Multi-platform (Web + API)
- Stores chat data

## Tech Stack
- Python (Flask)
- SQLite
- HTML/CSS

## Run
pip install flask  
python app.py
