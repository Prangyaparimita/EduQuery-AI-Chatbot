# \# EduQuery AI – Smart Student Support System for Universities with Multi-Platform Access 🤖

# 

# \## 📌 Description

# EduQuery AI is an intelligent web-based chatbot system designed to assist students with university-related queries such as admissions, fees, exams, placements, and academic information. The system uses a rule-based AI approach with keyword matching to understand user input and provide relevant responses instantly. It improves student support by reducing manual effort and ensuring quick query resolution.

# 

# \---

# 

# \## 🚀 Features

# \- 🤖 AI-powered chatbot (rule-based logic)

# \- 💬 Real-time chat interaction

# \- 🎯 Handles queries like exams, fees, admissions, placements

# \- 📱 Multi-platform support (mobile + desktop)

# \- 🗂 Stores user queries using SQLite database

# \- ⚡ Fast and lightweight system

# \- 📲 Callback request feature

# 

# \---

# 

# \## 🧠 AI Approach

# The system uses a \*\*rule-based AI model\*\* where user input is processed using keyword matching techniques. Based on predefined rules, the chatbot generates accurate and relevant responses. This approach ensures fast response time and simplicity while maintaining effectiveness for structured queries.
