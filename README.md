EduQuery AI – Smart Student Support System 🤖

Description
EduQuery AI is an intelligent chatbot system designed to assist students with university-related queries such as exams, fees, admissions, and placements. The system uses a rule-based Artificial Intelligence (AI) approach to understand user input and provide accurate responses instantly.

Features
- AI-powered chatbot using rule-based logic
- Handles student queries (fees, exams, admissions, placements)
- Real-time chat interaction
- Multi-device support (mobile and desktop)
- Stores user queries in SQLite database

AI Approach
This project implements a rule-based AI model where user queries are processed using keyword matching techniques. Based on predefined rules, the chatbot identifies user intent and generates appropriate responses. This makes the system fast, efficient, and suitable for structured query handling.

Technologies Used
Frontend:
- HTML
- CSS (Responsive design)
- JavaScript

Backend:
- Python
- Flask

Database:
- SQLite (auto-created during runtime)

AI Logic:
- Rule-based AI (Keyword Matching)

Other:
- REST API (communication between frontend and backend)
- GitHub (version control)

How to Run
1. Install Python  
2. Install Flask using: pip install flask  
3. Run the application: python app.py  
4. Open browser: http://127.0.0.1:5000/

Note
The database is automatically created when the application runs and stores user queries.
